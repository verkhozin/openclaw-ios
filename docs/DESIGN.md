# Design System — CLiOS

Дизайн-язык приложения. Общие принципы без точных значений — за конкретикой обращайся к `Theme.swift`.

---

## Философия

**Командный центр, не чат-приложение.** Информация должна считываться за долю секунды — одним взглядом на экран. Минимум шума, максимум плотности данных.

- Тёмная тема — единственная. Светлой нет и не будет.
- Глубина через слои, а не тени. Поверхности отличаются прозрачностью и блюром.
- Движение вместо статики. UI «дышит» — анимации пружинные, быстрые, тактильные.
- Цвет = значение. Случайного цвета не бывает.

---

## Цветовая модель

### Фон и поверхности
Иерархия глубины строится на слоях:

```
bg → surface → surfaceElevated
(самый тёмный)   →   (самый светлый)
```

Каждый следующий уровень чуть светлее предыдущего. Разделители — тонкие полупрозрачные линии (`border`).

### Текст
Три уровня контраста:

```
textPrimary → textSecondary → textMuted
(основной)     (подписи)       (плейсхолдеры)
```

### Акцент
Белый. Да, акцентный цвет — белый. Для кнопок, переключателей, активных элементов. Приглушённый вариант — полупрозрачный белый.

### Семантика
- **Успех** — зелёный (iOS system green)
- **Предупреждение** — жёлтый
- **Ошибка** — красный (iOS system red)

### Брендовые цвета сервисов
Карточки сервисов (GitHub, Stripe, Sentry и т.д.) используют фирменные цвета бренда в заголовке. Тело карточки — стандартный `surface`.

---

## Типографика

Только системный шрифт (`SF Pro` / `.system()`). Никаких кастомных шрифтов.

| Роль | Стиль | Насыщенность |
|------|-------|-------------|
| Заголовок экрана | title | bold |
| Заголовок секции | title2 | semibold |
| Основной текст | body | regular |
| Подписи | caption | regular |
| Код / технические данные | body, monospaced | regular |
| Код мелкий | caption, monospaced | regular |

## Поверхности и карточки

### Принцип «Стекло на стекле»
Основной паттерн — `ultraThinMaterial` поверх тёмного фона. Это даёт эффект матового стекла без тяжёлых теней.

### Карточки
- Скруглённые углы (два размера: стандартный и маленький)
- Тонкая обводка полупрозрачным `border`
- Заливка — `surface` или `surfaceElevated`
- Внутренний паддинг стандартный (`paddingM`)

### Сервис-карточки (Card Protocol)
Структура:
```
┌─ Цветной заголовок (бренд-цвет + иконка + название) ─┐
│                                                        │
│  Тело: заголовок, бейджи, метаданные, чеклисты,       │
│  markdown-контент, кастомные слоты                     │
│                                                        │
└────────────────────────────────────────────────────────┘
```

## Сообщения (Chat)

### Пузыри
- **Агент** (слева): светлый фон, скруглённые углы, минимальная тень. Форма — iMessage-стиль с «хвостиком».
- **Пользователь** (справа): контрастный фон (label color), белый текст, зеркальная форма хвостика.

### Стриминг
- Во время генерации — мигающий курсор
- Индикатор набора — анимированные точки (bounce)

## Анимации

### Пружины — основа всего
Никаких линейных или ease-in/out. Все переходы — `spring` с разной степенью упругости:

- **Быстрые** (кнопки, тапы): короткий отклик, высокое затухание
- **Средние** (раскрытие панелей, навигация): средний отклик
- **Мягкие** (фоновые элементы, шейдеры): длинный отклик, низкое затухание

### Паттерны появления
- Fade + scale + offset — стандартная комбинация для входа элементов
- Staggered delay — последовательное появление списков (каждый следующий элемент чуть позже)
- Клавиатура — специальная кривая, синхронизированная с iOS

### Шейдеры (Metal)
Фоны некоторых экранов — живые GPU-шейдеры (aurora, sky, plasma, rain, clouds). Это не декорация, а часть ощущения «живого» интерфейса.

### FluidGradient
CoreAnimation-система анимированных «блобов» для градиентных фонов. Отдельная от Metal-шейдеров.

## Лейаут

### Отступы
Три размера: маленький, средний, большой. Используются единообразно через `Theme.paddingS/M/L`.

### Структура экранов

```
┌─────────── Safe Area ───────────┐
│  Status bar fade gradient       │
├─────────────────────────────────┤
│                                 │
│  Content area                   │
│  (scrollable)                   │
│                                 │
├─────────────────────────────────┤
│  Bottom panel / input bar       │
│  (чёрный, отделён скруглением)  │
└─────────────────────────────────┘
```

### Навигация
4 таба: Chat, Tasks (Workspace), Dashboard, Settings. Tab bar — стандартный iOS.

## Иконки

SF Symbols. Всегда. Кастомные иконки — только для брендов сервисов.

## Принципы при разработке

1. **Все значения — из `Theme`**. Никогда не хардкодь цвета, шрифты, отступы.
2. **Adaptive colors** через Asset Catalog. Именованные цвета (`bg`, `surface`, etc.) определены как named colors.
3. **Анимация по умолчанию**. Если что-то появляется/исчезает — оно анимировано.
4. **Плотность > пустота**. Не раздувай отступы. Экран телефона маленький — используй каждый пиксель.
5. **Один источник правды** — `GatewayService`. Не создавай параллельные состояния для UI.
